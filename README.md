# twitch_reward_sender
Консольная утилита для отправки нескольких twitch наград

## Настройка запуска
### Получение токена авторизации
1. Открыть twitch и авторизоваться своей учеткой
2. Открыть консоль разработчика, нажав `F12`
3. Перейти в раздел `Network`
4. Для удобвства выбрать `Fetch/XHR`
5. В левой части выбрать любой `gql` запрос
6. Открыть раздел `Headers`
7. В части `Request Headers` найти `Authorization`
8. Сохранить значение, которое начинается на `OAuth ...`

### Настройка утилиты
1. Скачать `config.example.json`
2. Переименовать в `config.json` и сохранить в ту же директорию, где утила
3. В файле `config.json` заменить значение `OAUTH_TOKEN` на сохраненный токен

## Запуск
1. Запустить утилиту
2. Для управления используйте подсказки меню
